# comit W5
หัวข้อ Line bot แจ้งเตือนเพื่อสุขภาพ

ไอเดีย : การสร้างแพลตฟอร์มที่เข้ามาแก้ปัญหา การที่ผู้สงอายุหรือผู้ที่มีโรค ทั้งที่เป็นโรคประจำตัว หรือ โรคที่ต้องกินยาตามเวลา โดยจะทำหน้าที่เป็นผู้ช่วย ส่วนตัว ผ่าน แอพพิเคชัน Line

แหล่งข้อมูลอ้างอิง : App - Shop Healthcare - HDmall Website - https://hdmall.co.th/


แอปแจ้งเตือนกินน้ำ
•	การพัฒนาแอปพลิเคชันเตือนการดื่มน้ำสำหรับผู้สูงอายุ — งานวิจัยนี้มุ่งพัฒนาแอปเตือนดื่มน้ำสำหรับผู้สูงอายุ และประเมินประสิทธิภาพของแอป.[ssru]
•	Design and Development of a Mobile-Based Water Reminder Application on the iOS Platform — ศึกษาการพัฒนาแอปเตือนดื่มน้ำบน iOS โดยปรับคำแนะนำตามเพศ อายุ น้ำหนัก ส่วนสูง และระดับกิจกรรม.[ac]
•	The Use of Android-Based Applications “Drinking Alert” — พบว่าแอปเตือนดื่มน้ำแบบ Android ช่วยเพิ่มความรู้และทัศนคติของวัยรุ่นต่อการบริโภคน้ำได้อย่างมีนัยสำคัญ.[ac]
แอปแจ้งเตือนกินยา
•	การออกแบบและพัฒนาแอปพลิเคชันแจ้งเตือนการรับประทานยา — งานวิจัยมุ่งออกแบบแอปแจ้งเตือนกินยาสำหรับวัยทำงาน และศึกษาความพึงพอใจของผู้ใช้.[publication.npru]
•	ผลของแอปพลิเคชันแจ้งเตือนการรับประทานยาในผู้ป่วยโรคความดันโลหิตสูง — พบว่าหลังใช้แอป ความดันโลหิตเฉลี่ยลดลง และความพึงพอใจต่อแอปเพิ่มขึ้น.[he01.tci-thaijo]
•	การพัฒนากล่องยาอิเล็กทรอนิกส์แจ้งเตือนทานยาผ่านแอปพลิเคชันไลน์ — เป็นงานที่เชื่อมกล่องยากับ LINE เพื่อแจ้งเตือนเวลาใช้ยาและมีระบบเตือนอัตโนมัติ.[tci-thaijo]
